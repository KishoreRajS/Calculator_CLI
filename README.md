# Calculator CLI Application
## Project Overview

The Calculator CLI Application is a simple command-line program developed in Python that performs basic arithmetic operations. It provides a menu-driven interface where users can repeatedly perform calculations until they choose to exit.
This project demonstrates the use of Python fundamentals such as functions, loops, conditionals, and user input.

# Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero
- Menu-driven interface
- Runs continuously until the user exits

#Technologies Used
- Python 3.x
- VS Code / Any Python IDE
- Command Line Interface (CLI)
